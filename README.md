##### Getting Started
- clone the repository by running\
    `git clone https://github.com/GOlivierNation/math-magicians.git`
- navigate to the folder\
    `cd math-magicians`
- Install packages\
    `npm install`
- Run the app\
    `npm start`
- Eject the app\
    `npm run eject`
- Create build app\
    `npm run build`
- Test Linters app\
    `npm run check-linters`
